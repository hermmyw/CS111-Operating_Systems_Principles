who can take 3 values:
1. RUSAGE_SELF:
	- Gets the resource utilization for the parent: simpsh.
2. RUSAGE_CHILD:
	- Get the resource for all children/grandchildren/and so on.
3. RUSAGE_THREAD:
	- For the calling thread.

What we need:
- RUSAGE_SELF + RUSAGE_CHILD => 2 calls.

Lets say
simpsh invokes bash
	bash invokes cat and exits
		cat terminates (becomes zombie process)
cat's usage will not be included. 


Example for times:
 bash -c " ./example_getrusage | cat ; times;"
